# Семантические иконки Alpha 20

Основной реестр: `ru.citycore.gui.GuiIcon`. Все иконки имеют ванильный fallback. Цветные головы включаются параметром `gui.custom-heads` и не требуют ресурспака.

| Смысл | Цветная голова | Fallback |
| --- | --- | --- |
| Город | City Building | `BRICKS` |
| Экономика | Money Bag | `GOLD_INGOT` |
| Личный кошелёк | Wallet | `BUNDLE` |
| Казна | Digital Safe | `ENDER_CHEST` |
| Предприятие | Chest with Gold Coins | `CHEST` |
| Заявление | Mailbox | `WRITABLE_BOOK` |
| Документы | Box of Documents | `BOOK` |
| Связь | Mailbox | `PAPER` |
| Телефон | — | `CLOCK` |
| Звонок | — | `GOAT_HORN` |
| Рация | — | `SPYGLASS` |
| Мэрия | Crown | `GOLDEN_HELMET` |
| Госструктуры | Shield | `SHIELD` |
| Администрирование | Gear | `COMMAND_BLOCK` |
| Нефтяная отрасль | Oil Pump | `PISTON` |
| Нефть | Oil Bucket | `BUCKET` |

Хеши текстур взяты из публичного зеркала каталога Minecraft-Heads: <https://github.com/TheLuca98/MinecraftHeads>. В JAR сохраняются только идентификаторы официальных URL `textures.minecraft.net`, а не изображения или сторонний код.
