# Alpha 18 — конфигурация и миграция

## Новые параметры

| Путь | По умолчанию | Назначение |
|---|---:|---|
| `citizenship.reapply-cooldown-seconds` | `86400` | задержка после добровольного выхода перед новой заявкой |
| `industry.enabled` | `true` | запуск промышленного модуля; изменение требует перезапуска |
| `industry.cycle-seconds` | `3600` | длительность расчётного периода в реальном времени, минимум 30 секунд |
| `industry.max-catch-up-periods` | `3` | максимум пропущенных периодов за один запуск, диапазон 1–24 |
| `industry.default-lease` | `20.00` | аренда ресурсного участка за период |
| `industry.debt-suspend-threshold` | `1000.00` | долг, после которого производство блокируется |
| `industry.default-tax-bps` | `500` | начальный налог; 100 basis points равны 1% |
| `industry.max-tax-bps` | `2500` | верхняя граница ставки, доступной мэру |
| `industry.controller-material` | `LODESTONE` | блок контроллера; изменение требует перезапуска |
| `industry.levels.<1..3>.units-per-cycle` | `10/25/60` | плановая добыча уровня за период |
| `industry.levels.<1..3>.price-per-unit` | `25.00` | цена государственной закупки единицы нефти |
| `industry.levels.<1..3>.maintenance` | `40/120/360` | обслуживание объекта за период |
| `industry.levels.<1..3>.minimum-license-days` | `7/30/90` | минимальный срок лицензии соответствующего уровня |

Суммы указываются строками в основной валюте и переводятся в целые minor units. Изменение `database.*`, `economy.currency-scale`, `industry.enabled`, `industry.controller-material` или настроек LuckPerms через `/cc reload` отклоняется: эти параметры применяются только после полного перезапуска.

## Миграция схемы 4 → 5

Перед открытием пула и выполнением SQL создаётся `plugins/CityCore/backups/citycore-before-alpha18.db`. При наличии SQLite sidecar-файлов также копируются `-wal` и `-shm`. Существующая резервная копия не перезаписывается повторным запуском.

Миграция транзакционно:

- сохраняет все существующие таблицы городов, гражданства, предприятий, лицензий, счетов, ledger, эмиссии и аудита;
- добавляет вид деятельности предприятия с безопасным значением `GENERAL`;
- отделяет активное членство от неизменяемой истории ролей;
- добавляет процедуры передачи и административной отставки мэра;
- добавляет месторождения, участки, контроллеры, объекты, снимок инспекции, циклы и промышленную политику города.

Если любое действие миграции завершается ошибкой, транзакция откатывается, плагин отключается с диагностикой `Startup failure`, а резервная копия остаётся доступной для восстановления.
