# CityCore 0.1.0-alpha.19.0 — прогресс интерфейсного патча

Дата фиксации: 16.07.2026  
Источник требований: Мастер-ТЗ CityCore и утверждённые уточнения владельца проекта.

Обозначения: `[x]` — реализовано в исходниках; `[~]` — требует проверки собранного JAR на Paper; `[ ]` — не завершено.

## 15 обязательных требований

- [x] **1/15 — границы Мастер-ТЗ.** Патч меняет интерфейс существующих городов, предприятий, лицензий, финансов и нефтяной отрасли; новые поселения, телефон, события и иные идеи не добавляются скрыто.
- [x] **2/15 — назначение `/cc`.** Это техническая панель проекта, а не RP-телефон. Названия прямые и объясняют реальные системные функции.
- [x] **3/15 — три раздела главной.** На `/cc` оставлены ровно «Личное», «Город» и «Экономика», стоящие одним центральным рядом.
- [x] **4/15 — удаление мусора.** Раздел «Сервисы», фиктивные фильтры, заглушки и контекстная нижняя строка полномочий удалены.
- [x] **5/15 — полный визуальный словарь.** Доменные элементы больше не используют случайные колокола, кирпичи и цветной бетон: город, деньги, предприятия, документы, роли и промышленность имеют стабильные смысловые иконки.
- [x] **6/15 — цветные головы и fallback.** Для ключевых разделов добавлены цветные custom heads без ресурспака; `gui.custom-heads` включает их, а при отключении или ошибке остаётся понятный ванильный предмет.
- [x] **7/15 — чистое «Личное».** В разделе находятся только голова/идентичность игрока, личный кошелёк и личный статус. Предприятия, лицензии, выход и отставка отсюда убраны.
- [x] **8/15 — полный «Город».** Вступление, отзыв заявления, основание и его отзыв, выход гражданина, сложение полномочий чиновника, входящая/исходящая передача поста и отставка мэра находятся в одном смысловом разделе.
- [x] **9/15 — чистая «Экономика».** Личный кошелёк не повторяется. Основной экран показывает предприятия и их организационные счета; служебная казна появляется только внутри отдельного рабочего стола власти.
- [x] **10/15 — отдельные полномочия.** Переходы мэрии, госструктур и администрации полностью удалены из `/cc`; нижняя строка больше не меняется от роли.
- [x] **11/15 — рабочий стол `/ccm`.** Заявления жителей, роли, предприятия, регистрации, лицензии, казна и промышленная политика собраны горизонтальным рядом и возвращаются в мэрию.
- [x] **12/15 — рабочий стол `/ccg`.** Предприятия, регистрации, лицензии, жители и промышленные осмотры собраны отдельно и возвращаются в госструктуры.
- [x] **13/15 — рабочий стол `/cca`.** Основание городов, эмиссия, отставки/вакансии мэров и месторождения собраны отдельно. «Состояние системы» и «Перечитать настройки» из GUI удалены; соответствующие защищённые команды сохранены.
- [x] **14/15 — геометрия и навигация.** Рамка неприкосновенна, короткие наборы центрируются, списки идут слева направо, а «Назад / рабочая главная / Закрыть» занимают постоянные слоты `48/49/50`.
- [~] **15/15 — безопасность и приёмка.** PDC action ID, повторные capability-проверки, защита асинхронного обновления от устаревшего inventory и запрет извлечения предметов сохранены. Геометрия и реестр иконок покрыты тестами; остаются зелёная сборка GitHub Actions и полный серверный чек-лист.

## Решение по иконкам

Цветные головы используются только как визуальные прототипы предметов. Они не содержат действий сами по себе, не запрашивают права и не заменяют PDC. Текстуры задаются URL `textures.minecraft.net`, поэтому ресурспак не требуется. Набор хешей отобран из публичного каталога Minecraft-Heads; исходные названия и ссылки перечислены в `gui-icons.md`.

## Не входит в Alpha 19

- RP-телефон и приложения телефона;
- поселения и подчинённые территории;
- сезонные события, Blacksite и уничтожение карты;
- новые экономические отрасли поверх нефтяного среза;
- изменение схемы базы данных 5;
- удаление команд `/cc status` и `/cc reload` — они лишь убраны из `/cca`.
