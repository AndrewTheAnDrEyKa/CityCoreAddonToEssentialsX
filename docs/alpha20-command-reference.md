# CityCore Alpha 20 — команды игрока

## Навигация

| Полная команда | Короткая | Назначение |
|---|---|---|
| `/citycore` | `/cc` | Центр игрока |
| `/profile` | `/pf` | Личный профиль |
| `/city` | `/ct` | Город |
| `/business` | `/bz` | Предприятия |
| `/documents` | `/dc` | Документы и лицензии |
| `/citycoreadmin` | `/cca` | Администрирование |
| `/citycoremayor` | `/ccm` | Прямой вход в мэрию для совместимости |
| `/citycoregovernment` | `/ccg` | Прямой вход в работу чиновника для совместимости |

## Телефон

| Команда | Назначение |
|---|---|
| `/phone`, `/ph` | Открыть телефонный экран |
| `/phone call <игрок>` | Позвонить |
| `/ph <игрок>` | Короткий звонок |
| `/phone accept`, `/pha` | Принять |
| `/phone decline`, `/phd` | Отклонить |
| `/phone hangup`, `/phh` | Завершить или отменить |
| `/phone number` | Показать номер и модель |

## Рация

| Команда | Назначение |
|---|---|
| `/radio`, `/ra` | Переключить режим передачи |
| `/radio set <канал>` | Выбрать канал |
| `/radiofrequency <канал>`, `/rf <канал>`, `/radioset <канал>` | Быстро выбрать канал |
| `/radio list` | Список каналов |
| `/radio status` | Состояние |
| `/radio off` | Гарантированно выключить |
| `/radio send <текст>`, `/rc <текст>` | Разовая передача |

Команда `/r` не используется, чтобы не конфликтовать с ответом EssentialsX.

## Речь

- `текст` — локальная речь, по умолчанию 10 блоков;
- `! текст` — крик, по умолчанию 20 блоков;
- при активной рации сообщение также получают включённые рации на том же канале;
- при активном звонке сообщение также получает собеседник;
- телефон имеет приоритет над рацией.
