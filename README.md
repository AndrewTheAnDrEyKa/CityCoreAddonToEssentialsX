# CityCore 0.1.0-alpha.15.0

Чистая реализация ядра CityCore для Paper 1.21.8. Проект не продолжает повреждённый GUI-код alpha.14.9.

## Требования

- Java 21+
- Paper 1.21.8
- EssentialsX 2.21.2+
- Vault 1.7.3

LuckPerms не обязателен. Технические права используют Bukkit permissions, игровые городские роли хранятся CityCore.

## Сборка

```bash
gradle clean build
```

Готовый файл: `build/libs/CityCore-0.1.0-alpha.15.0.jar`.

## Уже реализовано в новом фундаменте

- SQLite, пул соединений, внешние ключи и версионированная миграция;
- выделенная очередь хранения — SQL не исполняется в игровом потоке;
- профили игроков;
- таблицы городов, гражданства, внутренних счетов и двойной записи;
- строгая интеграция Vault/EssentialsX Economy;
- единый GUI listener, действия через PDC и явные маршруты;
- главная, профильная, экономическая и административная панели;
- безопасный отказ запуска при отсутствии БД или Economy provider;
- предупреждение об offline-mode.

## Статус

Это первый компилируемый инкремент alpha.15.0, а не готовый релиз для публичного сервера. Следующий инкремент подключает транзакционные сценарии создания города, гражданство, казначейство и журнал аудита к уже созданной схеме.
